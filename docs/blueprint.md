# **App Name**: Dualogue

## Core Features:

- AI Chat Simulation: Simulate a conversation between two AI agents with configurable personalities and topics.
- AI Personality Configuration: Define the personality traits, background, and goals for each AI agent using a text field tool that informs the language model's approach.
- Topic Selection: Allow users to choose a topic or provide a starting prompt for the AI conversation.
- API Key Integration: Provide a secure field for users to enter their API key for the language model service. Accesses and validates the key.
- Chat Log Display: Display the ongoing conversation in a clear, chronological format, distinguishing between the two AI agents.
- Conversation Controls: Implement controls to start, stop, reset, load, and save the AI conversation.
- Model Parameter Tuning: Expose temperature, output token limit, and exchange count as tuning parameters, as illustrated in the picture.

## Style Guidelines:

- Primary color: Soft periwinkle (#B2BEB5) to create a calm and sophisticated feel.
- Background color: Very light gray (#F0F0F5), a desaturated periwinkle, provides a neutral backdrop.
- Accent color: Muted mauve (#9A869C) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a clean and modern interface.
- Simple, minimalist icons to represent conversation controls and AI personalities.
- A split-screen layout, with the chat log on one side and AI control hub on the other.
- Subtle animations for new messages and parameter adjustments.
